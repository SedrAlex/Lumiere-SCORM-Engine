What is SCORM?

SCORM (Shareable Content Object Reference Model) is a set of technical standards for e-learning software products. It defines how online learning content and Learning Management Systems (LMSs) communicate with each other.

Key features of SCORM include:

  • Content packaging standards
  • Launch and communication protocols
  • Data model for tracking learner progress
  • Sequencing rules for content navigation
