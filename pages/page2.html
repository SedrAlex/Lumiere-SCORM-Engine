SCORM Standards and Versions

SCORM has evolved over time with different versions offering enhanced capabilities. This page outlines the key differences between SCORM versions.

SCORM Version Comparison

Feature SCORM 1.2 SCORM 2004
Release Year 2001 2004 (with editions through 2009)
API Name API API_1484_11
Sequencing Basic Advanced (with conditional rules)
Navigation Limited Enhanced
Status Tracking lesson_status completion_status and success_status
Time Format HH:MM:SS ISO 8601 (e.g., PT1H30M5S)
